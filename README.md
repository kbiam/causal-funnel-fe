# 📘 Frontend - Quiz App

This is the **React frontend** for the Quiz Application. It allows users to start a quiz, answer questions, and view results, with session handling and history tracking.

---

## 🚀 Features

- Start a new quiz session
- Navigate through questions
- Submit answers in real-time
- View final score and result summary
- Track history of past quizzes
- Uses **AppContext** for managing global state (like session, user info, etc.)
- Responsive UI

---

## 🧰 Tech Stack

- **React.js**
- **React Router**
- **Context API (AppContext)**
- **Fetch API** for backend communication
- **LocalStorage** for storing session data
- **CSS / Tailwind / Bootstrap** (depending on your styling)
